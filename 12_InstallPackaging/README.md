# logingen - генератор системных логинов по имени пользователя
- Инструмент адаптации к окружению - Autotools
- Реализована сборка динамической библиотеки
- Реализован перевод сообщений
- Реализованы unit-тесты
- Реализована manpage
- Реализована выгонка документации из исходников
## Сборка:
```bash
autoreconf --install --force
./configure --prefix=DIR
make
```
Здесь `DIR` - каталог, в который установится программа.

## Установка
После сборки команда `make install` установит программу в `DIR/bin`.

## Использование
Посмотреть manpage: `man login-gen`.
Чтобы просмотреть HTTP-страницу с документацией на `localhost:8000`, нужно выполнить команду `make http`.

Поддерживается запуск программы в русской локали.

## Запуск unit-тестов:
```bash
make check
```
